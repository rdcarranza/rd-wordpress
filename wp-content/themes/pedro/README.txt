== Description ==  

**Pedro Portfolio** is a modern WordPress theme tailored for UI & UX designers, web developers, and freelancers.  
With a fully responsive layout and customizable options, it's designed to showcase your work beautifully.  
The theme features a user-friendly options panel, enabling you to make changes without any coding knowledge.  
It also includes a sleek homepage layout and supports various plugins to enhance your site.  
Ideal for professionals, startups, and creative agencies looking to impress potential clients.  

== Installation ==  

1. In your WordPress admin panel, go to **Appearance > Themes** and click **Add New**.  
2. Click **Upload Theme** and select the theme's `.zip` file. Click **Install Now**.  
3. After installation, click **Activate** to use your new theme.
